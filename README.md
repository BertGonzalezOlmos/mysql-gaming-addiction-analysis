# 🎮 Gaming Addiction Analysis with MySQL

## 📌 Project Overview

This project demonstrates practical SQL skills using a Gaming Addiction dataset. The objective is to practice database creation, data exploration, and analytical queries commonly used by Data Analysts.

The project covers SQL concepts from beginner to advanced level, making it an excellent portfolio project.

---

## 📂 Dataset

The dataset contains information about gamers, including:

* Demographics
* Gaming habits
* Daily playtime
* Monthly spending
* Mental health indicators
* Addiction scores
* Productivity metrics
* Behavioral clusters
